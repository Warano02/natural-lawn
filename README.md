# Natural Lawn Installation

A professional showcase website for Natural Lawn Installation, a company specialising in lawn laying, wild flower meadows and grounds maintenance across Sussex, Surrey and surrounding areas. Built with Next.js 16, Tailwind CSS v4 and TypeScript.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Configuration

In `app/globals.css`, make sure you have the color variables defined:

```css
@theme {
  --color-primary: #1a5c2a;
  --color-secondary: #6aaf2e;
}
```

## Images

Place all images inside `public/assets/images/` and the logos at `public/logo.png` and `public/logo-large.png`.

## Production

```bash
npm run build
npm start
```
